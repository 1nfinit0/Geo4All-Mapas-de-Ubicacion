# Mapas temáticos y de ubicación

Los mapas temáticos y de ubicación son herramientas visuales esenciales para representar datos geoespaciales de manera efectiva así como su contextualización en un marco geopolítico.

- **Mapas Temáticos**: Estos mapas se centran en un tema específico o variable, como la densidad de población, el clima, la distribución de recursos naturales, entre otros. Utilizan colores, símbolos y patrones para destacar las diferencias y tendencias en los datos representados.

- **Mapas de Ubicación**: Proporcionan una representación visual de la ubicación geográfica de un lugar o conjunto de lugares. Son útiles para identificar la posición relativa en el contexto geopolítico y hasta histórico donde se encuentre la información que se asocia.

## Características Clave

- **Visualización Clara**: Ambos tipos de mapas deben ser claros y fáciles de interpretar para un usuario sin conocimientos técnicos avanzados.
- **Contextualización Geopolítica**: Los mapas deben incluir elementos como fronteras, ciudades principales y otros puntos de referencia para situar la información en un contexto geopolítico adecuado.
- **No están sujetos a normativa de producción**: Estos mapas no requieren cumplir con normativas específicas de producción cartográfica, lo que permite mayor flexibilidad en su diseño y presentación.
- **Contienen**: títulos, leyendas, escalas y la fuente de datos para mejorar la comprensión del mapa.

Para esta sesión de trabajo, nos enfocaremos en el proceso de creación de un mapa temático y de ubicación, esta guía servirá como referencia para la generación de cualquier otro tipo de mapa similar. El uso de los datos presenten no contempla la generación de datos propios, sino la representación visual de los datos existentes en un formato geoespacial adecuado. Este último punto, sobre la generación de nueva información geoespacial a partir de datos georreferenciados y no georreferenciados, se abordará en sesiones futuras.
